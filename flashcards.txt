Every Linux filesystem traces its roots to which directory?::/ (root)
Which directory contains files pertaining to the initial boot of the system::/boot
Which directory contains critical executable files?::/bin
Which directory contains executable files runnable only by root?::/sbin
Which directory contains libraries?::/lib
Which directory hosts the bulk of a linux machines programs and program files?::/usr
Which directory hosts files that a system administrator installs locally?::/usr/local
Which directory houses files related to X?::/usr/X11R6
Which directory is intended for ready-made packages not installed with the OS?::/opt
Which directory houses all user's personal data?::/home
Which directory is root's home directory?::/root
Which directory contains transient files of various types, usually for logs and print spools?::/var
Which directory is intended for temporary use?::/tmp
Which directory is intended for temporarily mounting a filessytem?::/mnt
Which directory houses device files?::/dev
Which directory houses a virtual filesystem used to provide kernel-provided information?::/proc
